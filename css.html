Giới thiệu

CSS (Cascading Style Sheets) là ngôn ngữ định dạng kiểu dáng được sử dụng để thiết kế và trình bày các trang web. Nó giúp bạn kiểm soát giao diện và bố cục của các phần tử HTML trên trang web, tạo ra các thiết kế trực quan và hấp dẫn. Dưới đây là tất cả những gì bạn cần biết về CSS.

Mẫu định dạng CSS:

<style>
                h1 {color:red;}
                h1 {border: 2px solid blue;}
                p {text-intent:15px;}
                </style>

Mục đích và Cấu trúc

  • Mục đích:
    • Bộ chọn (Selectors): Chọn các phần tử HTML để áp dụng kiểu.
    • Thuộc tính (Properties): Xác định các khía cạnh cần thay đổi (ví dụ: color, font-size).
    • Giá trị (Values): Xác định cài đặt cho thuộc tính (ví dụ: red, 16px).
    • Quy tắc CSS (CSS Rules): Kết hợp bộ chọn, thuộc tính và giá trị.
  • Cấu trúc:
    • Tách biệt nội dung (HTML) và trình bày (CSS).
    • Kiểm soát giao diện: màu sắc, phông chữ, bố cục, hiệu ứng.
    • Đảm bảo tính nhất quán trên toàn bộ trang web.
    • Tạo thiết kế đáp ứng, phù hợp với nhiều thiết bị.

Cách sử dụng

  • CSS nội tuyến (Inline CSS): Viết trực tiếp trong thẻ HTML bằng thuộc tính "style".
  • CSS nhúng (Internal/Embedded CSS): Viết trong thẻ <style> trong phần <head> của trang HTML.
  • CSS liên kết (External CSS): Viết trong file .css riêng biệt và liên kết đến trang HTML bằng thẻ <link>. (Khuyến khích sử dụng cách này)

Các khái niệm chính

  • Bộ chọn CSS:
    • Bộ chọn phần tử (Element selectors): Chọn các phần tử HTML dựa trên tên thẻ (ví dụ: p, h1, div).
    • Bộ chọn lớp (Class selectors): Chọn các phần tử HTML có thuộc tính "class" cụ thể (ví dụ: .my-class).
    • Bộ chọn ID (ID selectors): Chọn phần tử HTML có thuộc tính "id" duy nhất (ví dụ: #my-id).
    • Bộ chọn thuộc tính (Attribute selectors): Chọn các phần tử HTML có thuộc tính cụ thể.
    • Bộ chọn giả (Pseudo-classes): Chọn các phần tử dựa trên trạng thái của chúng (ví dụ: :hover, :active).
    • Bộ chọn giả phần tử (Pseudo-elements): Chọn các phần con của phần tử (ví dụ: ::before, ::after).
  • Mô hình hộp (Box Model):
    • Nội dung (Content): Nội dung thực tế của phần tử.
    • Đệm (Padding): Khoảng trống giữa nội dung và đường viền.
    • Đường viền (Border): Đường bao quanh phần tử.
    • Lề (Margin): Khoảng trống bên ngoài đường viền.
  • Bố cục (Layout):
    • Flexbox: Mô hình bố cục linh hoạt, giúp sắp xếp các phần tử dễ dàng.
    • Grid: Mô hình bố cục lưới, cho phép tạo bố cục phức tạp
    • Float: Sắp xếp các phần tử nằm cạnh nhau.
    • Position: Định vị các phần tử trên trang.
  • Đơn vị:
    • px (pixel): Đơn vị tuyệt đối.
    • em, rem: Đơn vị tương đối, dựa trên kích thước phông chữ.
    • %: Đơn vị phần trăm.
    • vw, vh: đơn vị tương đối với chiều rộng và chiều cao của viewport.

Lợi ích

  • Kiểm soát giao diện chi tiết.
  • Giảm kích thước file HTML.
  • Cải thiện khả năng tiếp cận.
  • Tăng tốc độ tải trang.
  • Giúp trang web trở nên chuyên nghiệp và đẹp mắt hơn.
